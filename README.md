## 🚀 Running the Project Locally

To run this project on your local machine, follow these steps (you might need some API keys such as OpenWeatherAPI, Amazon SES SMTP, etc to run the app)

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
    cd your-repo-name
    ```

2.  **Install dependencies:**
    ```bash
    # Install PHP dependencies
    composer install

    # Install JavaScript dependencies
    npm install
    ```

3.  **Environment Setup:**
    ```bash
    # Create a copy of the .env.example file
    cp .env.example .env
    ```
    Open the `.env` file and configure your database (`DB_*` variables) and mail (`MAIL_*` variables) credentials.

4.  **Generate Application Key:**
    ```bash
    php artisan key:generate
    ```

5.  **Run Database Migrations:**
    ```bash
    php artisan migrate
    ```

6.  **Compile Assets:**
    ```bash
    npm run dev
    ```

7.  **Start the development server:**
    ```bash
    php artisan serve
    ```

You should now be able to access the application at `http://127.0.0.1:8000`.
